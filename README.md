# Digital Drifting

### Tracing the Convergence of Digital Access Across U.S. Regions, 2013–2023

This repo contains all experimental code for the ECON4331W writing project. 
- data_processing: this folder contains the data processing pipeline for the project.
- analysis: this folder contains all data analysis files. 
- convergence: this folder contains the convergence modeling files for the experiments. 
- webapp: this folder contains the R Shiny web application we developed for the writing project. 
- data: you should create this folder after you cloned this project to store all the data files needed for the analysis and modeling.
